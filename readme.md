                                        Проект: Bank Console Application
                                            
                                            Описание приложения
                    Данный проект представляет собой консольное приложение от Clever-Bank.
        Во время работы приложения просходит постоянная проверка (каждые 30 сек) является ли сегодняшний день последним
    в месяце. Когда наступает последнее число, происходит (один раз в этот день) начисление процентов на баланс каждому
    клиенту Clever-Bank. Значение процента хранится в файле конфигурации (src/main/resources/config.properties),
    по умолчанию установлено "percentMoney=1". При проверке и начислении работают отдельные потоки, что позволяет
    пользователю независимо использовать свои возможности в приложении.
        В приложении пользователю доступны следующие возможности:
        * вход в приложение по своему уникальному ID номеру (соответствует номеру ID в базе данных, табл. "clients")
        * регистрация нового клиента Clever-Bank с автоматическим заведением ему банковского счета)
        * просмотр информации по всем своим банковским счетам (независимо в каком банке они были заведены)
        * пополнение денежных средств любого своего банковского счета
        * снятие денежных средств с любого своего банковского счета
        * перевод денежных средств другому клиенту Clever-Bank (с любого своего счета относящегося к любому банку)
        * перевод денежных средств клиенту другого банка (с любого своего счета относящегося к любому банку).

                                            Запуск приложения
      1. Перед запуском приложения необходимо в файле конфигурации (src/main/resources/config.properties)
       указать локальные данные (dbUser- пользователь, dbPass- пароль, dbName- название вашей базы данных)
      2. В точке старта "main()" в методе startApp(DataBase.runType) выбрать значение объекта DataBase (Enum)
        * DataBase.NEW - при старте приложения загрузжаются новые данные в базу данных. Если до этого в базе данных
         уже имелись таблицы (clients, banks, transactions, bank_accounts) они будут перезаписаны.
        * DataBase.OLD - при старте приложение будет обращаться к существующим данным в базе данных.
          ВНИМАНИЕ! - если данные (таблицы "clients, banks, transactions, bank_accounts") отсутствуют на момент запуска,
          приложение выведет сообщение о пустой базе дынных (EMPTY DATA  BASE) и закончит свою работу.

                                        Работа приложения
    Как только приложение будет запущено, в консоли среды разработки (например IDEA) будет выведено меню авторизации
                                     
                                    РУКОВОДСТВО ПО МЕНЮ АВТОРИЗАЦИИ

                        -------------------Clever-Bank--------------------
                        -----------------AUTHORISED MENU--------------------
                        1. Log in to Clever-Bank by ID
                        2. Become a new client at CleverBank
                        0. Exit the application

    На данном этапе требуется ввести с консоли 1, 2 или 0.
    ВНИМАНИЕ!- при некорректном вводе будет выведено сообщение "INCORRECT INPUT! Enter a number"

        1- вход в приложение по уникальному ID клиента Clever-Bank.
        2- регистрация нового клиента в Clever-Bank
        0- выход из приложения. Программа завершит свою работу.

        При вводе 1: (Log in to Clever-Bank by ID)
    При выборе пункта меню "1" пользователю приложения будет предложено ввести уникальный номер ID клиента Clever-Bank

                        Enter your unique ID number

        Если пользователь является клиентом Clever-Bank и его данные находятся в базе данных, пользователь успешно войдет
    в главное меню. В случае отсутствия клиента Clever-Bank с таким ID, который ввел пользователь, в консоль будет
    выведено сообщение "Is not a client of Clever-Bank:(" и произойдет возврат в меню авторизации.
       * Для просмотра возможностей приложения введите уникальный ID клиента Strukov Alexandr, номер ID = 1.

    а также создается привязанный к нему новый банковский счет в Clever-Bank с балансом равным 0 BYN. Происходит переход
